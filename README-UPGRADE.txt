BOBU SITE UPGRADE

1. Upload index.html, style.css, script.js and the assets folder to the root of your GitHub repository.
2. Keep your existing "litepaper" folder, or upload BOBU_Litepaper_v1.0.pdf there.
3. For a moving hero background, add an MP4 named:
   assets/bobu-space.mp4
4. Recommended video:
   - 1920x1080
   - 8–20 seconds
   - muted/no audio
   - compressed under 8–12 MB
   - seamless loop
5. GitHub Pages will update after the commit is deployed.

The design still works without the MP4 because hero-poster.png is used as a fallback image.
